Fully Responsive MMA WordPress Theme.

== Description ==

Fully Responsive MMA WordPress Theme. With its powerful design aesthetics, it captures the raw intensity
and passion of martial arts, ensuring your website stands out in the arena. Whether you run an MMA gym,
offer training programs, or manage fighters, ShogunPro is your perfect cornerman for online success.
Features include Built-in Homepage Slider (Displayed Only on a Static Front Page), W3C markup validated,
Search Engine Optimized, Header Social Icons (Facebook, Twitter, LinkedIn, Instagram, Tumblr, YouTube,
Pinterest), Sidebar, Multiple Browsers Support, Translation Ready and much more.
